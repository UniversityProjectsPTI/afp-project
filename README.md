# Beadandó projekt

## Alapkövetelmények:
Mielőtt lehúzod a projektet telepítsd ezeket:
- Git: https://git-scm.com/downloads
- Node: https://nodejs.org/en/

## Projekt futtatása:
- git clone
- npm install
- npm run

## Branch védelem:
- Védett mappa: main

## Push folyamata:
- Melyik branchen vagyunk? **git branch**
- main? ha igen akkor **git checkout -b újbranch**
- git add .
- git commit -m "üzenet"
- git push
